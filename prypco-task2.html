PRYPCOVisual Assets
4 / 14

Visual & graphic asset localization.

Not all images are universal — three categories require action.

Replace

Category 1 — Images with embedded text

  • Property price banners
  • Promo cards with 'AED 500'
  • App store screenshots
  • Onboarding illustrations with labels

Replace with Arabic version or use text overlay approach.

Mirror

Category 2 — Directional icons & UI elements

  • Back arrow (← becomes →)
  • Progress indicators (L→R becomes R→L)
  • Carousel pagination dots
  • Timeline components

CSS transform: scaleX(-1) on arrow/chevron icons.

Audit

Category 3 — Cultural & religious imagery

  • People imagery (conservative dress)
  • Financial lifestyle photos
  • Holiday/seasonal visuals
  • Trust badges and regulatory logos

Audit all imagery with a cultural reviewer on D3.

Machine translation

3 · Human Review

Native speaker audit

Arabic speaker reviews ar.json in audit UI. Corrects mistranslations. Flags cultural issues.

4 · Codemod

jscodeshift transform

AST transform replaces 'Invest Now' with t('cta.invest') across all files. Zero manual find-and-replace.

5 · RTL + Assets

Layout & mirrors

react-native-rtl-layout. CSS logical properties. Directional icons mirrored. Images with text flagged.

PRYPCOQuality Assurance
10 / 14

How we validate quality fast.

Three layers of QA — automated first, human last, all within D8–D9.

Automated (D8 — hours)

  • Detox RTL suite — screenshot tests for all 20 key screens in AR locale. Pixel-diff against EN baseline.
  • Playwright (web) — headless browser tests: truncation, overflow, input direction, form validation in AR.
  • ESLint i18n rule — catches any new hardcoded string committed after the sprint. Fails the build.
  • BrowserStack — real device matrix: iPhone 14 Arabic, Samsung S23 Arabic, Chrome Arabic on Android.

Native Speaker (D3 + D9 — 2hr sessions)

  • String audit (D3) — reviews 100% of ar.json. Corrects mistranslations. Flags financial/legal terms (DFSA, SPV, IBAN).
  • Screen walkthrough (D9) — walks through all 20 key screens on a real Arabic iOS device. Signs off on each screen.
  • Cultural review — imagery, colors, idioms, Shariah-relevant copy audited by MENA reviewer.
Zero screen passes QA without explicit sign-off from both automated tests and native speaker review.
